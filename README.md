# React Dashboard Project

I am creating this project to follow along with a [tutorial series](https://www.youtube.com/watch?v=wYpCWwD1oz0&ab_channel=EdRoh) I am following to learn more about React and Material UI.

## Setting up the project

I added the following dependencies to set up the project:

Material UI

- @mui/material
- @mui/x-data-grid
- @mui/icons-material

Emotion

- @emotion/react
- @emotion/styled

Full Calendar

- @fullcalendar/core
- @fullcalendar/daygrid
- @fullcalendar/timegrid
- @fullcalendar/list

nvio

- @nivo/core
- @nivo/pie
- @nivo/line
- @nivo/bar
- @nivo/geo

Miscellaneous

- react-router-dom
- react-pro-sidebar
- formik
- yup
